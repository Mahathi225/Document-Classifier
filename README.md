# Document Classifier

## Overview
This project is a machine learning-based document classifier that categorizes text documents into predefined categories.

## Categories
- sci.space
- rec.autos
- comp.graphics

## Technologies Used
- Python
- Scikit-Learn
- TF-IDF Vectorization
- Multinomial Naive Bayes

## Result
The model achieved approximately 90% accuracy.

## Sample Prediction

Input:
NASA plans a new mission to Mars

Output:
sci.space
